## Описание
Тестовая сборка для просмотра стека технологий, тестовых задачек

Новое: 
- (27.11.25) Пылесосим устройство на другие установленные приложения, контрольные суммы.
- (26.05.25) горизонтальный слайдер фоток с индикатором
- (26.05.25) MediaPlayer и ExoPlayer для воспроизведения роликов/музыки (видео по url, музыку достаю локально)

## Технологии
- **Язык:** Kotlin
- **UI:** Jetpack Compose
- **Навигация:** Jetpack Navigation
- **Зависимости:** Dagger-Hilt
- **Сетевые запросы:** Retrofit
- **Кэширование:** Room
- **Изображения:** Coil
- **Асинхронность:** Coroutines
- **Воспроизведение:** MediaPlayer и ExoPlayer
